# Dalamud Plugins

> A small collection of FFXIV Dalamud plugins (client-side quality-of-life).

## Installation

1. In Dalamud: **Settings -> Experimental -> Custom Plugin Repositories**
2. Add this URL:
   ```
   https://raw.githubusercontent.com/VelvetFFXIV/DalamudPlugins/main/pluginmaster.json
   ```
3. **Save** -> open the Plugin Installer -> install any plugin below.

## Plugins

### XIVIgnore
*Virtually extends the in-game ignore list (client-side).*

Client-side virtual extension of the FFXIV ignore list: additionally mute and hide players, with categories, notes and auto-expiry. UI available in English and German.

[Repository, details &amp; issues](https://github.com/VelvetFFXIV/XIVIgnore)
